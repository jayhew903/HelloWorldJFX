Appointment Setting Application
The purpose of this system is to be able to set up appointments for our customers

Author: Jayde Hewitt
Contact: jmalars@wgu.edu
Version: 1
Date: August 2023

IDE: IntelliJ Community 2020.01
JDK Version: JDK SE 11.0.4
JavaFX Version: JavaFX-SDK-11.0.2

What to do:
Launch the Application
Please use the below information to log into the program:
Username: test
Password: test

Once logged in you will be able to add, update and delete customers and appointments.
The first screen you will view is the Appointments page.
This page will display all the current appointments for the company.
From here selecting an appointment then “modify apt” will allow updates to be made to that appointment.
Selecting “add apt” will allow a new appointment to be added.
***PLEASE NOTE ALL APPOINTMENT TIMES ARE IN MILITARY TIME***
EX. 1:00 AM = 01:00; 1:00 PM = 13:00
Selecting an appointment then “delete appointment” will delete the current selected appointment.
There is also an option to select “view customers” That will display the customer’s table.
Customers page will have the same functions as the Appointments Page.
Another option is “reports” which will display the reports page.
Reports page will display the total number of appointments based on the selected month and Type.
It will also count the total number of customers and the total number of appointments in the database.
It will display the appointments based on the selected contact in a table.

Additional Report:
The additional report chosen was to count the total number of customers and appointments in the database.
The number of total customers will increase as customers are added in.
The number of total appointments will increase as appointments are added.

MySQL Connector: mysql-connector-java-8.0.25
